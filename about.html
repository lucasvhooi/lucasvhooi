ABOUT ME

Hey! I am Lucas. I am a 22 year old collage graduate with a degree in graphic design with the specialization 3D art. I have a good understanding of the game asset workflow and I enjoy making my own small game scenes with self made models and shaders. I have done a lot of work in Unity and have a good understanding of how this engine works. Trying to get better at technical art is also one of my goals for the time to come.Lately I have been busy with web design and using HTML, CSS and JavaScript so I can start taking my design skills to the web. In my spare time I work at a high-end clothing store. I love working with the different brands and trends in the fashion world. I also own a small clothing brand where I express my view on what a premium colourful streetwear brand should look like. Using Adobe Photoshop, Illustrator and InDesign I love to create logos and other kinds of content for social media and websites.

This website is programmed using HTML and CSS by me using VS Code
