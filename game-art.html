Here are three models that I made using Substance Designer. I used small variations of the textures to blend them together in Unity to avoid repetition and create a more natural look. I really like the way you build up materials in substance designer and I wanted to improve my skills with creating own textures from the ground up. PIANO MODEL

×

This is a 1:1 model of my own keyboard. Its a low poly game ready asset. I started with measuring my keyboard and noting everything down. After that I blocked out the model using basic shapes in Maya. When I had finished the blockout I started adding all the details.

For texturing I needed to unwrap the model. for texturing I used Substance Painter. I created a few diffrent materials that I used to texture the model. I made the final renders using the render in Substance Painter.

LEGENDS OF ESSOLIS

×

Legends Of Essolis is a small game project I was working on after my internship. I wanted to get some C# experience by creating a small playable demo scene. I also wanted to try and expand my knowledge of shader building in Unity. My idea was to create a low poly stylized puzzle adventure game. With the focus on using the environment with the puzzles.

All models and shaders are made by me with exception of the rocks. Using Unity ShaderGraph I created a water, vegetation and toon shader. With the toon shader I wanted to create a more stylized look to the game. Using C# I created a 3rd person character controller, a simple pickup system a waypoint system and a inventory system and a say night cycle. For the firefly's and fire I used particle systems. All models are made using Maya and textures are made using a colorgrid system in photoshop. I also used some optimisation tools like LOD meshes and occlusion culling.

I perosnaly think that the level turned out great. A lot of people really liked the style of the level and really wanted to see more about the game. I learend a lot about optimisation and making shader in Unity. Overall it was a really positive experience!
